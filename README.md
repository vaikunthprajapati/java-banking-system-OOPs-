# Java Banking System

A console-based banking application developed in Java to demonstrate Object-Oriented Programming concepts.

## Concepts Used

* Abstraction
* Encapsulation
* Inheritance
* Polymorphism
* Classes and Objects

## Features

* Create Accounts
* Deposit Money
* Withdraw Money
* Display Account Information
* Savings Account Support

## Technologies

* Java
* OOP Principles
